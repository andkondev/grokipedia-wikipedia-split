Grokipedia + Wikipedia Split View

Official support and privacy page for the browser extension that opens Grokipedia and Wikipedia side by side in a single tab.

Overview Web Store Availability Privacy Policy Support and links

Overview

This extension has one purpose: help users compare related pages on grokipedia.com and en.wikipedia.org side by side. It can be launched from the toolbar button or right-click context menu on supported pages.

When exact title matching is not available, the extension falls back to search pages so users can continue navigating in split view.

Privacy Policy

This Privacy Policy applies to the Grokipedia + Wikipedia Split View browser extension.

Data processed by the extension

  • Page URLs on supported domains to determine whether split view can be opened and to keep both panes in sync.
  • On-page article title text (for example, a page <h1>) to resolve Wikipedia targets when needed.

Data stored by the extension

  • Sync extension storage: user preferences such as context-menu enabled state and optional pane-sync setting. On supported browsers, these settings may sync across the user's browser profiles or devices when browser sync is enabled.
  • Local extension storage: split-view runtime state such as split tab ID, current Wikipedia URL values, and one-time pin reminder flags.
  • localStorage on grokipedia.com: split-pane width preference (grokipediaSplitWidthPercent) so the selected pane ratio can be restored quickly on page load.

Network requests

  • The extension sends requests only to grokipedia.com and en.wikipedia.org to resolve or validate target article pages used by split view.
  • No third-party analytics, advertising, or telemetry endpoints are used by the extension.

What the extension does not collect

  • No personally identifiable information such as name, address, phone number, or government identifiers.
  • No authentication secrets such as passwords, tokens, or security answers.
  • No payment, health, precise location, or personal communications data.

Data sharing and selling

Developer does not sell user data and does not transfer user data to third parties except as required to load and request pages from the supported sites named above.

Data retention and deletion

Stored settings/state remain in browser storage (including Chrome extension storage and the split-width localStorage value on grokipedia.com) until changed by the user, cleared by the browser, or removed when the extension is uninstalled.

Security

Data handled by the extension is limited to functional operation and is stored using browser-managed extension storage or local page storage as described above. No separate backend database is used by this extension.
